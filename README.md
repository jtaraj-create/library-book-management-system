# library-book-management-system

This is a simple front-end web project for managing library books.

## Features

- Add new books
- Edit existing books
- Delete books
- Search books by title, author, or ID
- Filter books by category
- Clear all book records
- Save data using browser localStorage

## Technologies Used

- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap

## How to Run

1. Download or clone the project.
2. Open `index.html` in a browser.
3. Use the buttons to add, edit, delete, search, or filter books.

## Notes

The project uses localStorage, so the books are saved in the browser even after refreshing the page.
